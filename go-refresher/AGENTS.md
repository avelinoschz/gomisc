# AGENTS

This subtree intentionally differs from the root repository guidance in the following ways:

- Treat `go-refresher/` as a structured Go skills-refresher track, not as a loose archive of snippets.
- Exercises may span multiple files when that makes backend-style practice clearer.
- From `03-testing` onward, tests are expected and are part of the intended workflow.
- Exercises after the first one may intentionally include incomplete starter code and TODOs.
- Notes and exercises in this area may be slightly more production-oriented than the repo's usual snippet style.
