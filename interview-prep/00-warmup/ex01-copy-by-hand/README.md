# Warmup Ex01: Copy By Hand

Goal:

- regain rhythm with basic Go syntax
- write `struct`, methods, slices, loops, and errors again without friction

What to do:

1. Do not edit this file first.
2. Open `main.go`.
3. Copy it by hand into a temporary file or rewrite it without looking at each line.
4. Run the program and make sure you understand each method.

Things to memorize:

- pointer receiver
- `append`
- index-based iteration
- early return with an error
- linear search over a slice

Command:

```bash
go run ./interview-prep/00-warmup/ex01-copy-by-hand
```
