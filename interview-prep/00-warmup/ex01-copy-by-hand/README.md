# Warmup Ex01: Copy By Hand

Objetivo:

- recuperar ritmo con sintaxis basica de Go
- volver a escribir `struct`, metodos, slices, loops y errores sin friccion

Que hacer:

1. No edites este archivo primero.
2. Abre `main.go`.
3. Copialo a mano en un archivo temporal o reescribelo sin mirar cada linea.
4. Corre el programa y verifica que entiendes cada metodo.

Puntos a memorizar:

- receiver con puntero
- `append`
- iteracion con indice
- retorno temprano con error
- busqueda lineal en un slice

Comando:

```bash
go run ./interview-prep/00-warmup/ex01-copy-by-hand
```
